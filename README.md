# 📊 Сроки годности — Telegram Бот

Бот принимает Excel-файлы по 4 магазинам (АСБ, КАМ, ПОБ, СКЛ) и генерирует итоговый файл **Сроки годности**.

---

## 🚀 Деплой за 5 шагов

### 1. Создать бота в Telegram
1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Придумайте имя и username
4. Скопируйте токен вида `1234567890:ABCdef...`

### 2. Форкнуть репозиторий на GitHub
1. Нажмите **Fork** в правом верхнем углу
2. Или создайте новый репозиторий и загрузите эти файлы

### 3. Подключить Railway
1. Зайдите на [railway.app](https://railway.app) → **Login with GitHub**
2. **New Project** → **Deploy from GitHub repo**
3. Выберите ваш репозиторий

### 4. Добавить переменную окружения
В Railway: **Variables** → **New Variable**:
```
BOT_TOKEN = ваш_токен_от_BotFather
```

### 5. Готово! 🎉
Railway автоматически установит зависимости и запустит бота.
Проверьте в **Deployments** — статус должен быть **Active**.

---

## 💬 Как пользоваться ботом

| Команда   | Действие                        |
|-----------|---------------------------------|
| `/start`  | Начать работу                   |
| `/status` | Посмотреть, какие файлы загружены |
| `/reset`  | Сбросить текущую сессию         |

**Процесс:**
1. Нажмите `/start` → **Начать**
2. Выберите магазин (кнопка)
3. Отправьте `.xlsx` файл для этого магазина
4. Повторите для оставшихся 3 магазинов
5. После 4-го файла бот автоматически пришлёт готовый файл

> Файлы можно отправлять в любом порядке.
> Если в имени файла есть АСБ/КАМ/ПОБ/СКЛ — бот определит магазин автоматически.

## ⚙️ Локальный запуск (для разработки)

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ваш-username/srok-bot.git
cd srok-bot

# 2. Установить зависимости
npm install

# 3. Создать .env файл
cp .env.example .env
# Вставить токен в .env

# 4. Запустить
npm run dev
```

## ⚠️ Технические детали

- Файлы хранятся **в памяти** (не на диске) — исчезают после перезапуска
- Если Railway перезапустит сервер во время загрузки — придётся загрузить файлы заново
- Бот работает в режиме **polling** (не нужен HTTPS и webhook)
- Поддерживает **одновременную работу** нескольких пользователей
